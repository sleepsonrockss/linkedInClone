# 💼 LinkedIn Clone

A static front-end clone of LinkedIn built with HTML and CSS, recreating
core UI elements of the LinkedIn interface.

## 🛠️ Built With

- HTML
- CSS
- JavaScript

## 📁 Structure

```
├── index.html    # Main LinkedIn feed page
├── style.css     # Stylesheet
└── script.js     # JavaScript interactions
```

## 🚀 Getting Started

Clone the repo and open `index.html` in your browser — no dependencies or
build steps needed.

```
git clone https://github.com/sleepsonrockss/linkedInClone.git
```
